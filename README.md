# junseongpyo.github.io

Personal academic website of Junseong Pyo.

Built with plain HTML/CSS and [htmx](https://htmx.org/) for partial loading.

## Structure

```
index.html        # Layout, styles, TOC, publications data
styles.css        # Shared site styles
about.html        # About, Education
publications.html # Publications (data-driven, JS)
research.html     # Research Experience
work.html         # Work Experience
figures/          # Paper figures
```

## Development

```bash
npx live-server --port=8080
```
