This is a sample post written in Markdown. Add new posts by creating files in
the `posts/` directory with front matter like `title`, `date`, `description`,
`tags`, and `permalink`.

## Writing workflow

1. Create a new Markdown file in `posts/`.
2. Add front matter at the top of the file.
3. Run `npm run dev` locally to preview.
4. Run `npm run build` before publishing.
